Webアプリケーション デプロイ手順書

1. GitHubアカウントの作成

  1. GitHubにアクセス
  2. 「Sign up」をクリック
  3. メールアドレス、パスワード、ユーザーネームを入力
  4. メール認証を完了

2. GitHubリポジトリの作成

  1. GitHubにログイン後、右上の「+」マーク → 「New repository」をクリック
  2. Repository nameに「mate-ranking」と入力
  3. 「Public」を選択
  4. 「Add a README file」にチェックを入れる
  5. 「Create repository」をクリック

3. Gitの設定（Git Bashを使用）

  1. スタートメニューまたはWindowsの検索で「Git Bash」を探して起動
  2. Git Bashで以下のコマンドを実行： 
    git config --global user.name "あなたのGitHubユーザー名"
git config --global user.email "あなたのメールアドレス"

4. ローカルプロジェクトのGitHub連携

  1. Git Bashでプロジェクトフォルダに移動： 
    cd /c/Users/murakami/mate
  2. Gitリポジトリを初期化： 
    git init
git add .
git commit -m "Initial commit"
  3. GitHubリポジトリと連携（URLは作成したリポジトリのものに置き換え）： 
    git remote add origin https://github.com/あなたのユーザー名/mate-ranking.git
git branch -M main
git push -u origin main

5. Renderへのデプロイ

  1. Renderにログイン
  2. 「New +」→「Web Service」を選択
  3. GitHubリポジトリを連携（初回は「Configure account」から連携設定が必要）
  4. 作成したリポジトリ「mate-ranking」を選択
  5. 以下の設定を確認：
    • Name: mate-ranking
    • Region: お好みの地域
    • Branch: main
    • Root Directory: ./
  6. 「Create Web Service」をクリック

デプロイ後の確認

  1. デプロイが完了するまで5-10分程度待機
  2. ステータスが「Live」になったことを確認
  3. 生成されたURLにアクセスして動作確認

注意点

  • 無料プランでは15分間アクセスがないとスリープ状態になります
  • スリープからの復帰に30秒程度かかることがあります
  • GitHubにプッシュすると自動的にRenderに反映されます
